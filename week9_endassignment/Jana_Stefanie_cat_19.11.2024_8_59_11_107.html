2 Write create statements for the database

Write create statements to create a database called catdb.
Create a table cat, for cat data. Table columns are:
  • number integer
  • name varchar(12)
  • length integer
  • weightKg integer
  • yearOfBirth integer
All columns are mandatory. The primary key is number.
Add two insert statements into the create statements for testing.
